# VitalCore Health — AWS Cloud Migration Simulator

An interactive cloud migration simulation dashboard built with React.
Simulates an 8-hospital network's migration from on-premise infrastructure to AWS.

## Group Members
Yupeng Dai · Christopher Kong · Dakota McElligott · Katrina Paulson · Bryan Rogers

## Course
CSCI S-176 Advanced Cloud Technology Management · Harvard Summer School 2026

## 9 Simulation Panels
- P1 · Executive Financial Dashboard — TCO, NPV, ROI, 5-year cash flow table
- P2 · Value Realization — waterfall chart, benefit breakdown, color-coded return zones
- P3 · Cloud Governance Dashboard — HIPAA compliance, security radar, org structure
- P4 · Decision Rights & Operating Model — approval workflows, SLA dashboard
- P5 · DevOps & Implementation Simulator — DORA metrics, automation, deploy frequency
- P6 · Resilience & Availability Dashboard — uptime curve, risk heatmap, availability scorecard
- P7 · AI Investment Explorer — ROI curve, marginal returns, efficiency indicator
- P8 · Organizational Adoption Dashboard — S-curve, resistance index, readiness score
- P9 · Product Diffusion & Market Expansion — Bass model, CAC vs LTV, market penetration

## Extra Credit
- EC1 · Cloud Cost Management — AWS Cost Explorer simulation, FinOps policy lever, anomaly detection (Demo Mode)

## AI Board Advisor
Persistent sidebar powered by the Claude API — answers board-level questions in plain English using live simulation context.

## Built With
React, Recharts, Claude API (claude-sonnet-4-6)

- This project was implemented in React/Vite rather than Streamlit. React was selected to provide a richer, more responsive interactive dashboard while fulfilling the same functional requirements outlined in the assignment.
