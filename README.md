# VitalCore Health — AWS Cloud Migration Simulator

An interactive cloud migration simulation dashboard built with React.
Simulates a hospital network's migration from on-premise infrastructure to AWS.

## Panels
- Executive Summary
- Financial (TCO, NPV, ROI + Value Realization)
- Governance (HIPAA Compliance + Decision Rights)
- Operations (DevOps/DORA + Resilience & HA)
- Innovation (AI Investment Explorer)
- Adoption (Org Adoption + Market Diffusion)
- Cost Management — Extra Credit (AWS FinOps Simulator)

## AI Board Advisor
Powered by Claude API — ask board-level questions in plain English.

## Built With
React, Recharts, Claude API (claude-sonnet-4-6)
